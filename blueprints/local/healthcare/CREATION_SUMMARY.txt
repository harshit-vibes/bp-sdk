CLINICAL DESIGN RATIONALE
================================================================================

PROBLEM ADDRESSED
- 30-day hospital readmissions cost $15B+ annually
- Common causes: Poor discharge planning, medication errors, missed follow-ups
- Evidence: Comprehensive discharge coordination reduces readmissions 20-30%

WHY THREE WORKERS?
- Discharge Summary: Complex synthesis requiring good documentation skills
- Medication Reconciliation: Safety-critical, needs specialized pharmaceutical knowledge
- Follow-up Scheduling: Requires care planning and barrier identification
- Separation of concerns improves quality and allows independent optimization

WHY THESE TEMPERATURES?
- Manager (0.3): Balanced reasoning for orchestration logic
- Discharge Summary (0.3): Consistent, professional documentation
- Medication Reconciler (0.2): VERY LOW for safety-critical drug interactions
- Follow-up Scheduler (0.3): Balanced for care planning decisions

WHY MANAGER AGENT?
- Ensures correct workflow order (safety → medication → follow-up)
- Integrates outputs for consistency checking
- Performs quality assurance before package is complete
- Handles edge cases (complex patients, barriers)
- Prevents workers from being called in wrong order

WHY NOT JUST ONE AGENT?
- Single agent would create generic output without specialization
- Multiple agents allow focused instructions and optimization
- Manager orchestration ensures workflow consistency and safety
- Easier to test, maintain, and improve individual specialists
